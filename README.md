# EmailToCardbase
Windows-сервис для мониторинга почты, конвертации CSV и загрузки в БД через Cardimp.exe.

Card Import Service

Автоматический конвейер обработки карт: мониторинг почты → извлечение CSV из ZIP → конвертация в TXT → загрузка в базу данных через Cardimp.exe.

Работает как фоновая служба Windows. Настроен через config.ini, ведет логи и историю обработанных писем.

Создан с использованием Python и ИИ-ассистента как инструмента ускорения разработки. Полный цикл: от идеи до продакшена — 40 часов.

## Возможности
- Проверка почтового ящика по IMAP
- Автоматическое извлечение ZIP-вложений
- Конвертация CSV в TXT нужного формата
- Запуск утилиты импорта Cardimp.exe
- Работает как служба Windows
- Ведёт логи и историю обработанных писем
- Настраивается через config.ini

📁 Структура проекта
```text
D:\UCS\IMPORT\
├── mail.py                  # Проверка почты
├── main.py                  # Обработка ZIP
├── convert_last12.py        # Конвертер CSV → TXT
├── email_service.py         # Служба Windows
├── config.ini               # Настройки
├── processed_emails.json    # История писем
├── install_service.bat      # Установка службы
├── uninstall_service.bat    # Удаление службы
├── Cardimp.exe              # Утилита импорта в БД
├── attachments\             # Вложения из писем
├── OLD\                     # Архив обработанных
└── logs\                    # Логи работы
```
