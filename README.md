#  EmailToCardbase

Автоматический конвейер импорта карт: мониторинг почты → извлечение CSV из ZIP → конвертация в TXT → загрузка в базу данных.

![Python](https://img.shields.io/badge/Python-3.12+-blue)
![Platform](https://img.shields.io/badge/Platform-Windows%20Service-lightgrey)
![Status](https://img.shields.io/badge/Status-Production-green)

## О проекте

Служба Windows, которая без участия человека забирает вложения из писем, конвертирует CSV в требуемый формат TXT и запускает утилиту импорта `Cardimp.exe`. Полностью настраивается через `config.ini`, ведёт логи и историю обработанных писем.

## Возможности

- Проверка почтового ящика по IMAP
- Автоматическое извлечение ZIP-вложений
- Конвертация CSV в TXT нужного формата (`convert_last12.py`)
- Запуск утилиты импорта `Cardimp.exe` с параметрами
- Работа в фоне как служба Windows (`email_service.py`)
- Логирование и история обработанных писем
- Гибкая настройка через `config.ini`

## Структура проекта

```text
├── mail.py                  # Проверка почты (IMAP)
├── main.py                  # Обработка ZIP + вызов конвертера и Cardimp.exe
├── convert_last12.py        # Конвертер CSV → TXT
├── email_service.py         # Служба Windows (основной цикл)
├── config.ini               # Настройки (почта, пути, параметры)
├── processed_emails.json    # История обработанных писем
├── install_service.bat      # Установка службы
├── uninstall_service.bat    # Удаление службы
├── Cardimp.exe              # Утилита импорта в БД (сторонняя)
├── attachments\             # Вложения из писем (рабочая папка)
├── OLD\                     # Архив обработанных файлов
└── logs\                    # Логи работы службы
Установка и запуск
Установите Python 3.12+ и необходимые зависимости

Настройте config.ini (почта, пароль, пути)

Запустите install_service.bat от администратора

Служба запустится автоматически

Особенность
Проект полностью создан с использованием ИИ-ассистента как инструмента ускорения разработки. Полный цикл — от идеи до продакшена — занял 40 часов.
